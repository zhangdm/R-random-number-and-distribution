\begin{abstract}
  常见产生各种分布的随机函数，这里将给出函数及其语法规则，若干案例！

  给出的函数有：
  \footnote{r - 生成相应分布的随机数;
            d - 生成相应分布的密度函数;
            p - 生成相应分布的累积概率密度函数;
            q - 生成相应分布的分位数函数。
  }

  \begin{itemize}

	\item 均匀分布, runif
	\item 正态分布，rnorm
	\item 二项分布，rbinom
    \item 指数分布，rexp
    \item 贝塔分布，beta
    \item 柯西分布，rcauchy
    \item F分布，rf
    \item 伽马分布，rgamma
    \item 几何分布，rgeom
    \item 逻辑分布，rlogis
    \item 负二项分布，rnbinom
    \item 泊松分布，rpois
    \item 威尔考克森分布，rwilcox
\end{itemize}
% 关键字

\textbf{keywords:} R语言  ; 随机函数; CTEX
\end{abstract}
